Prefer `decl_search`, `reference_search`, or `symbol_context` for any code-symbol query in Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript files — they follow imports, aliases, and re-exports that regex misses. Use `read_file` only for literal text (strings, comments, config keys) or files outside Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript. Read a bounded byte slice from one workspace file and return its sha256 receipt. Use `read_file` once the graph (or a free-form `grep`) has produced a path and span.
