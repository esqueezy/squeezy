Prefer `decl_search`, `reference_search`, or `symbol_context` for bare-name symbol queries in indexed source — they follow imports and re-exports that regex misses. Use `read_file` for literal text or file types the graph does not index. Read a bounded byte slice from one workspace file and return its sha256 receipt. Each `content` line is prefixed with its 1-based absolute line number and a tab (cat -n format); `start_line` carries that number for the first line. Pass `offset`/`limit` to fetch only the section grep flagged; reading whole files when you need one body wastes tokens. For a symbol_id from a graph packet, `read_slice` with `span_kind=body` is cheaper.
