Prefer `decl_search`, `reference_search`, or `symbol_context` for any code-symbol query in Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript files — they follow imports, aliases, and re-exports that regex misses. Use `grep` only for literal text (strings, comments, config keys) or files outside Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript. Search text files under a workspace path. Respects .gitignore by default; set include_ignored=true only when ignored files are intentionally needed. Use output_mode=count or files_with_matches for broad exploration before reading content.
