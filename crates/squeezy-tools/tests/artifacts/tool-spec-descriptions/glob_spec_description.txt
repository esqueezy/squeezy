Prefer `decl_search`, `reference_search`, or `symbol_context` for any code-symbol query in Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript files — they follow imports, aliases, and re-exports that regex misses. Use `glob` only for literal text (strings, comments, config keys) or files outside Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript. List workspace file paths matching a glob without reading file contents. Respects .gitignore by default; set include_ignored=true only when ignored paths are intentionally needed.
